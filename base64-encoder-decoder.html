About Base64

What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data in ASCII string format.

Common Uses

Email attachments, data URIs, storing binary in JSON/XML, and encoding credentials.

Important Notes

Base64 is not encryption! It's encoding that can be easily decoded by anyone.

Size Increase

Base64 increases data size by approximately 33% due to 6-bit character representation.
